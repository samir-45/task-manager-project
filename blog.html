Question-2: What is the difference between innerHTML, innerText, and textContent ?

The is a huge difference between innerHTML, innerText, and textContent. InnerHTML is like this represents the intire content of a tag including html code also. Conversely, innerText just represents the inner text of any html element. And textContent represents the all existing content inside a tag

Question-3: What is event delegation in the DOM?

Event delegation is a pattern that handles events by attaching a single event listener to a parent element instead of adding listeners to each individual child element
